# Инструкция по добавлению пользователя в Linux

## 1. Создание пользователя

Для создания нового пользователя используйте команду `useradd`:

```bash
sudo useradd -m -s /bin/bash -G users username
```

**Параметры команды:**
- `-m` — создает домашний каталог для пользователя
- `-s /bin/bash` — устанавливает оболочку bash
- `-G users` — добавляет пользователя в группу users
- `username` — имя нового пользователя (замените на нужное)

## 2. Установка пароля

После создания пользователя установите ему пароль:

```bash
sudo passwd username
```

Система запросит ввод нового пароля дважды. Пароль должен содержать 6-8 символов, включая буквы разного регистра, цифры или знаки препинания.

## 3. Выдача прав sudo

Чтобы пользователь мог выполнять команды с правами администратора, добавьте его в группу sudo:

```bash
sudo usermod -aG sudo username
```

## 4. Проверка

Переключитесь на созданного пользователя и проверьте права:

```bash
su - username
sudo whoami
```

Команда `sudo whoami` должна вернуть `root`, что подтверждает наличие прав администратора.

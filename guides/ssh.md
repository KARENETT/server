
## 🔐 SSH ключи

### Генерация SSH ключей

SSH ключи обеспечивают безопасный вход на сервер без ввода пароля.

#### Рекомендуемый способ (ED25519)

ED25519 - современный алгоритм, более безопасный и быстрый, чем RSA:

```bash
ssh-keygen -t ed25519 -C "ваш_email@example.com"
```


#### Альтернативный способ (RSA 4096)

Для старых систем используйте RSA с размером ключа 4096 бит:

```bash
ssh-keygen -t rsa -b 4096 -C "ваш_email@example.com"
```

#### Процесс генерации

1. **Выбор расположения**: Нажмите Enter для сохранения в `~/.ssh/id_ed25519` (по умолчанию)
2. **Парольная фраза**: Введите надёжную парольную фразу для дополнительной защиты (рекомендуется)[web:216][web:217]
3. **Готово**: Созданы два файла:
   - `id_ed25519` - приватный ключ (НИКОГДА не передавайте его)
   - `id_ed25519.pub` - публичный ключ (этот можно передавать)

### Добавление SSH ключа на сервер

#### Способ 1: Автоматический (ssh-copy-id)

Самый простой способ:

```bash
ssh-copy-id -i ~/.ssh/id_ed25519.pub user@server_ip
```


**Замените**:
- `user` - ваше имя пользователя на сервере
- `server_ip` - IP-адрес сервера

#### Способ 2: Ручной

Если `ssh-copy-id` недоступен[web:217]:

**1. Скопируйте публичный ключ**
```bash
cat ~/.ssh/id_ed25519.pub
```

**2. Подключитесь к серверу**

```bash
ssh user@server_ip
```

**3. На сервере создайте директорию и файл**

```bash
mkdir -p ~/.ssh
chmod 700 ~/.ssh
nano ~/.ssh/authorized_keys
```

**4. Вставьте скопированный публичный ключ и сохраните (Ctrl+O, Enter, Ctrl+X)**
**5. Установите правильные права**
```bash
chmod 600 ~/.ssh/authorized_keys
```

#### Проверка подключения

Попробуйте подключиться без пароля:
```bash
ssh user@server_ip
```

**После запуска скрипта setup.sh SSH порт изменится на 33556**:

```bash
ssh -p 33556 user@server_ip
```

### Алиасы и конфигурация SSH (~/.ssh/config)

**Создайте файл для упрощения подключений:**

```bash
nano ~/.ssh/config
```

**Добавьте следующие строки:**

```
Host myserver
HostName server_ip
Port 33556
User <username>
IdentityFile ~/.ssh/id_ed25519

Host github
HostName github.com
User git
IdentityFile ~/.ssh/id_ed25519
```
